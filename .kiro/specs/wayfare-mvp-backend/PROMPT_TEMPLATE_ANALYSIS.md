# WayFare批注Prompt模板分析

## 概述

本文档分析WayFare系统中三种批注类型的Prompt模板设计，验证其是否符合教育心理学原则和学习科学最佳实践。

## 1. Explanation模板（费曼技巧）

### 模板内容

```
用户选中的文本：
{selected_text}

相关上下文：
{context}

请用简单易懂的语言解释这段内容，包括：
1. 核心概念是什么
2. 用类比或例子说明
3. 为什么这个概念重要

保持简洁，不超过200字。
```

### 教育心理学原理

**费曼技巧（Feynman Technique）**
- ✅ **简化语言**：要求"简单易懂的语言"
- ✅ **类比和例子**：明确要求使用类比或例子
- ✅ **概念重要性**：帮助学生理解学习动机

**认知负荷理论（Cognitive Load Theory）**
- ✅ **字数限制**：200字限制避免认知过载
- ✅ **结构化**：三个明确的要点降低外在认知负荷

**建构主义学习理论**
- ✅ **连接已有知识**：通过类比连接学生已有认知
- ✅ **意义建构**：解释"为什么重要"帮助建立意义

### 优势

1. 符合费曼技巧的核心原则
2. 结构清晰，易于LLM理解和执行
3. 字数限制确保输出简洁
4. 三个维度全面覆盖概念理解

### 改进建议（未来）

- 可以根据概念复杂度动态调整字数限制
- 可以添加"检查理解"环节

## 2. Question模板（启发性问题）

### 模板内容

```
用户选中的文本：
{selected_text}

相关上下文：
{context}

请提出2-3个启发性问题，帮助学生：
1. 理解概念的本质
2. 联系已有知识
3. 思考应用场景

每个问题简短有力。
```

### 教育心理学原理

**苏格拉底式提问（Socratic Questioning）**
- ✅ **引导思考**：通过问题引导而非直接告知
- ✅ **深度理解**：关注"概念本质"
- ✅ **批判性思维**：鼓励主动思考

**元认知理论（Metacognition）**
- ✅ **知识连接**："联系已有知识"促进元认知
- ✅ **迁移应用**："应用场景"培养迁移能力

**最近发展区理论（Zone of Proximal Development）**
- ✅ **适度挑战**：问题设计在学生能力范围内
- ✅ **支架作用**：问题作为认知支架

### 优势

1. 三个维度覆盖理解、连接、应用
2. 问题数量适中（2-3个）
3. 强调"简短有力"确保问题质量
4. 符合探究式学习原则

### 改进建议（未来）

- 可以根据学生水平调整问题难度
- 可以添加"开放式"vs"引导式"问题类型选择

## 3. Summary模板（要点总结）

### 模板内容

```
用户选中的文本：
{selected_text}

相关上下文：
{context}

请总结这段内容的核心要点：
1. 主要观点（1-2句话）
2. 关键细节（2-3个要点）
3. 与上下文的关系

保持简洁，不超过150字。
```

### 教育心理学原理

**信息加工理论（Information Processing Theory）**
- ✅ **分层处理**：观点→细节→关系的层次结构
- ✅ **工作记忆优化**：150字限制符合工作记忆容量

**图式理论（Schema Theory）**
- ✅ **知识组织**：结构化总结帮助建立图式
- ✅ **关系建立**："与上下文的关系"促进图式整合

**生成效应（Generation Effect）**
- ✅ **主动加工**：总结过程促进深度加工
- ✅ **记忆增强**：生成性学习提高记忆保持

### 优势

1. 三层结构符合信息层次
2. 字数限制更严格（150字）确保精炼
3. 强调上下文关系促进知识整合
4. 明确的数量指导（1-2句话、2-3个要点）

### 改进建议（未来）

- 可以添加"关键词提取"环节
- 可以生成思维导图式的结构化总结

## 系统提示词分析

### 内容

```
你是WayFare学习助手，帮助学生理解和掌握知识。

你的职责是：
1. 使用费曼技巧，用简单易懂的语言解释复杂概念
2. 通过启发性问题引导学生深入思考
3. 提炼核心要点，帮助学生建立知识框架

你的回答应该：
- 简洁明了，不超过200字
- 贴近学生的认知水平
- 结合具体例子和类比
- 鼓励主动思考和探索
```

### 设计优势

1. **角色定位清晰**：学习助手而非知识灌输者
2. **方法论明确**：费曼技巧、启发式提问、要点提炼
3. **质量标准**：简洁、适龄、具体、鼓励
4. **教育理念**：建构主义、主动学习

### 符合的教育原则

- ✅ 学生中心（Student-Centered Learning）
- ✅ 主动学习（Active Learning）
- ✅ 认知支架（Cognitive Scaffolding）
- ✅ 个性化学习（Personalized Learning）
